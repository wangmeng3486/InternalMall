# 企业内部福利商城 - 前端架构信息

|- **更新日期**: 2026-02-26
|- **技术栈**: Vue 3 + Vue Router + Pinia + Element Plus + Axios + Vite
|- **状态**: 开发中

## 目录结构

```
frontend/
├── src/
│   ├── api/
│   │   ├── index.js            # axios实例、拦截器、基础接口(apiHealth/apiAuth/apiUser/apiAdmin)
│   │   ├── admin.js            # 管理员接口集合(商品CRUD/分类CRUD/订单/用户/系统/上传/统计)
│   │   ├── cart.js             # 购物车接口集合
│   │   └── product.js          # 商品接口: apiProduct(用户端) + apiAdminProduct(管理端)
│   ├── components/
│   │   ├── admin/
│   │   │   ├── CategoryManage.vue  # 分类管理对话框
│   │   │   ├── ProductForm.vue     # 商品新增/编辑表单对话框
│   │   │   └── ProductList.vue     # 商品列表(搜索/表格/分页/状态切换/删除)
│   │   ├── layout/
│   │   │   └── AdminLayout.vue     # 管理后台布局(侧边栏+顶部栏)
│   │   └── product/
│   │       └── ProductCard.vue     # 商品卡片(图片/收藏/加入购物车)
│   ├── router/
│   │   └── index.js            # 路由定义
│   ├── stores/
│   │   ├── auth.js             # 认证状态(登录态/用户信息/localStorage持久化)
│   │   ├── cart.js             # 购物车状态(列表/统计/增删改查/批量操作)
│   │   ├── product.js          # 管理端商品状态(商品列表/分类/分页/过滤/排序)
│   │   └── userProduct.js      # 用户端商品状态(商品浏览/分类/收藏/分页/过滤)
│   ├── views/
│   │   ├── HomeView.vue              # 首页(登录入口/测试账号展示)
│   │   ├── ConnectivityTest.vue      # 连通性测试页面
│   │   ├── auth/
│   │   │   └── LoginView.vue         # 登录页(用户/管理员切换/快速填充/表单验证)
│   │   ├── admin/
│   │   │   ├── DashboardView.vue     # 管理仪表板(统计卡片/系统信息) [已实现]
│   │   │   ├── ProductManageView.vue # 商品管理(集成ProductList/ProductForm/CategoryManage) [已实现]
│   │   │   ├── OrderManageView.vue   # 订单管理 [空壳]
│   │   │   ├── UserManageView.vue    # 用户管理 [空壳]
│   │   │   └── SystemSettingsView.vue # 系统设置 [空壳]
│   │   ├── user/
│   │   │   ├── CartView.vue          # 购物车(全选/批量/数量调整/结算栏) [已实现]
│   │   │   ├── FavoritesView.vue     # 收藏夹(收藏列表/加入购物车) [已实现]
│   │   │   └── ProfileView.vue       # 个人资料(信息展示/导航侧栏) [已实现]
│   │   └── product/
│   │       ├── ProductBrowseView.vue  # 商品浏览(搜索/排序/分类过滤/分页) [已实现]
│   │       └── ProductDetailView.vue  # 商品详情(图片/规格/收藏/加购) [已实现]
│   ├── App.vue                 # 根组件(仅router-view)
│   └── main.js                 # 入口(Pinia/Router/ElementPlus注册)
├── index.html
├── package.json
└── vite.config.js              # 代理配置: /api和/uploads代理到后端
```

## 路由表

| 路径 | name | 组件 |
|------|------|------|
| `/` | home | HomeView.vue |
| `/products` | products | ProductBrowseView.vue |
| `/products/:id` | product-detail | ProductDetailView.vue |
| `/connectivity-test` | connectivity-test | ConnectivityTest.vue |
| `/login` | login | LoginView.vue |
| `/user/profile` | user-profile | ProfileView.vue |
| `/user/favorites` | user-favorites | FavoritesView.vue |
| `/user/cart` | user-cart | CartView.vue |
| `/admin/dashboard` | admin-dashboard | DashboardView.vue |
| `/admin/products` | admin-products | ProductManageView.vue |
| `/admin/orders` | admin-orders | OrderManageView.vue |
| `/admin/users` | admin-users | UserManageView.vue |
| `/admin/settings` | admin-settings | SystemSettingsView.vue |

## 模块依赖关系

- **LoginView** → auth.js(store) → api/index.js(apiAuth)
- **ProductBrowseView / ProductDetailView** → userProduct.js(store) → api/product.js(apiProduct)
- **CartView** → cart.js(store) → api/cart.js
- **FavoritesView** → userProduct.js(store) → api/product.js(apiProduct)
- **ProfileView** → auth.js(store) → api/index.js(apiUser)
- **ProductManageView** → product.js(store) → api/product.js(apiAdminProduct) + api/admin.js
- **DashboardView** → api/admin.js
- **AdminLayout** → auth.js(store) + router
